## Getting Started



### Prerequisites

Ensure you have Node.js installed

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ConverseScholar/hackharmony
   ```
2. cd into startup-nextjs-main
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the website 
   ```sh
   npm run dev
   ```
